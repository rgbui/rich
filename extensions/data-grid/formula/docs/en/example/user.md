
# User 说明

### id <a id='id'></a>
- **功能说明**: 用户的唯一标识符。
- **示例代码**: 
```
{name}.first().id // 输出: 1234567890
```

### name <a id='name'></a>
- **功能说明**: 用户的名字。
- **示例代码**: 
```
{name}.first().name // 输出: John Doe
```

### avatar <a id='avatar'></a>
- **功能说明**: 用户的头像链接。
- **示例代码**: 
```
{name}.first().avatar.url // 输出: https://example.com/avatar.png
```

### roles <a id='roles'></a>
- **功能说明**: 用户的角色列表。
- **示例代码**: 
```
{name}.first().roles // 输出: ["管理员"]
```
