# {name}
## 数字类型
 
### 函数Math.abs
取一个数的绝对值


#### 示例

```
Math.abs({name})

```

### 函数Math.cbrt
返回任意数字的立方根.

#### 示例

```
Math.cbrt({name})

```

### 函数Math.ceil
对数字进行上取会


#### 示例

```
Math.ceil({name})

```

### 函数Math.exp
返回 e 的 x 次幂（次方）Ex，其中 e = 2.718281... 是自然对数的基数。


#### 示例

```
Math.exp({name})

```

### 函数Math.floor
对数字进行下取会


#### 示例

```
Math.floor({name})

```

### 函数Math.pow
返回 x 的 y 次幂（ 次方 ）。

#### 示例

```
Math.pow({name}, 3)=={name}*{name}*{name} 

```

### 函数Math.sqrt
返回任意数字的平方根.

#### 示例

```
Math.sqrt({name})==3 //true
```
